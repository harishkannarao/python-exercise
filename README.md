# python exercise

Repository to explore python programming, testing and bundling of python programs

## Tools Required

* python `3.9`
* make `3.81`
* git `latest`
* pycharm `latest`

## Commands

### Install dependencies

    make init

### Update packages to latest

    make update
    
### Run tests

    make test
    
### Verify flake8

    make flake8
    
### Create requirements.txt

    make requirements
    
### Create distribution

    ./create_distribution.sh
